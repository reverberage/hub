# GitHub Accelerator

> Source: ``accelerator.github.com/``
> Status: [OK] Confirmed

---

## What It Is

- 10-week virtual program for open source maintainers
- Funding + mentorship + curriculum
- Focus: AI-based OSS projects with OSI-approved license
- Partner: GitHub Fund (M12 / Microsoft Venture Fund)

---

## Requirements

- Current maintainer/contributor of GitHub OSS project
- Age 18+
- Active GitHub profile
- In a GitHub Sponsors-supported region (Argentina is supported)
- NOT a GitHub/Microsoft employee
- Clear open source license

---

## Benefits

- Funding (amount not disclosed publicly; prior cohorts ~$10K-$20K)
- Access to GitHub Fund + M12 network
- 10-week curriculum: open source, AI, security, fundraising, business
