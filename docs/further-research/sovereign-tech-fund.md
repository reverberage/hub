# Sovereign Tech Fund (Germany)

> Source: ``sovereigntechfund.de/en``, ``sovereigntechfund.de/en/programs/fund``
> Status: [OK] Confirmed

---

## What It Is

- German government agency funding critical open-source digital infrastructure
- Multiple programs for FOSS projects

---

## Programs

| Program | Description | Amount |
|---------|-------------|--------|
| **Tech Fund** | Development grants for OSS infrastructure | **Min e50,000** |
| **Resilience** | Bug bounty + vulnerability prevention | Variable |
| **Fellowship** | Pays maintainers directly | Salary-like |
| **Standards** | Supports OSS in standards bodies | Pilot (Jun 2026-Jun 2027) |
| **Challenge** | Competition model for structural problems | Up to e300,000 |

---

## Tech Fund Requirements

- Minimum grant: e50,000
- FOSS license required
- "Prevalence" — widely used within/for other technologies
- Cannot be already funded by other public entities
