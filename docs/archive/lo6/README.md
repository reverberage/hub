# lo6 — Newsroom OS (demo)

**This is a preserved design artifact, not an active product.**

lo6 is a conceptual AI-powered newsroom operating system — an "Ableton Live for News."
It showcases one use case of the [reverberage](../../../README.md) toolkit:
how composable satellites (transcriber, verify, scout, transform) can be orchestrated
into a domain-specific workflow.

## What's here

- [**concept.html**](./concept.html) — Visual demo of the "Editorial Avant-Garde" design system
- [**docs/**](./docs/) — Design specifications (data model, agent interfaces, security, accessibility, frontend spec, funding plan)

## Status

This demo exists as a design reference. It is not an active product.
If a newsroom funder or partner expresses interest, lo6 can be revived
from these designs and built on top of the reverberage satellite stack.
